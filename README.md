# MeteoDash - Demo Escolar

Sistema meteorológico escolar con dashboard local y conexión Arduino.

## Descarga

Ve a la sección Releases y descarga:

MeteoDash_Demo_Escolar_v1.zip

## Uso

1. Extrae el ZIP.
2. Abre `1 - Instalar MeteoDash`.
3. Abre `2 - Abrir MeteoDash`.
4. Si el Arduino no aparece como COM, abre `3 - Arduino IDE Legacy Drivers`.

## Nota

No ejecutes el proyecto desde dentro del ZIP. Primero extrae la carpeta completa.
